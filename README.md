# Git常用命令
   ```bash
# 初始化仓库并设置远程链接
git init
git remote add origin <远程仓库的URL>
git remote -v # 查看远程仓库配置
git fetch --all # 获取所有分支代码
git add . # 获取所有分支代码
git commit -m "你的提交说明" # 提交变更（附带提交信息）
git push origin main # 推送代码到远程仓库主分支
git pull origin main # 从远程仓库拉取代码
```
